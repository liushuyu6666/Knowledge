# Materials
A wealth of learning resources are available for Spring Boot.

## Official Docs
1. To access the Spring Boot documentation, navigate to the official website at https://docs.spring.io/spring-boot/docs/ and select the latest release version. In the reference directory, you can find two versions of the documentation - PDF and HTML. For example, the HTML version of the Spring Boot 2.3.9 release can be found at https://docs.spring.io/spring-boot/docs/2.3.9.RELEASE/reference/html/index.html.

2. https://docs.spring.io/spring-boot/docs/current/reference/html/index.html

3. [Other Spring project](https://spring.io/projects)
